# Homebrew Tap for arcitems

This is a Homebrew tap for [arcitems](https://github.com/pdavlin/arcitems), a CLI tool for ARC Raiders item and quest lookup.

## Installation

```bash
brew tap pdavlin/arcitems
brew install arcitems
```

## Updating

```bash
brew upgrade arcitems
```

## Uninstall

```bash
brew uninstall arcitems
brew untap pdavlin/arcitems
```
